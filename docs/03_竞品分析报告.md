# 基于大模型的智能宠物诊疗系统竞品分析报告

**数据佐证版｜含图表｜含主流分析方法**  
**报告日期：2026-05-18**  
**分析对象：AI 宠物健康管理与辅助诊疗系统**

---

## 0. 报告摘要

本报告围绕“基于大模型的智能宠物诊疗系统”进行竞品分析，采用 **竞品分层法、功能矩阵、战略群组分析、波特五力模型、价值曲线分析、SWOT / TOWS 分析** 等广泛使用的商业分析方法。

综合判断：

1. **赛道不是空白，但尚未完全饱和。** 宠物电商、宠物医院、线上问诊、智能硬件、内容社区、通用大模型都已经占据部分用户场景，但多数玩家只覆盖单点需求，尚未形成“AI 分诊 + 宠物健康档案 + 兽医承接 + 线下医院 + 用药复诊 + 硬件预警”的完整闭环。
2. **最强直接竞品是京东健康宠物医院。** 京东健康已形成宠物互联网医院、7×24 在线问诊、线下医院转诊、宠物药品履约与 AI 宠物医生能力。其优势在医生、药品、履约、保险、医院网络和平台流量。
3. **线下医疗集团是专业承接型竞品。** 新瑞鹏等连锁宠物医疗集团拥有真实线下诊疗能力，优势是检查、诊断、治疗、手术、住院和专科能力，劣势是日常健康管理和线上高频触达较弱。
4. **智能硬件厂商是数据入口型竞品。** PETKIT 等硬件厂商掌握进食、饮水、如厕、活动等连续数据，但缺少执业兽医、线下医院和合规诊疗闭环。
5. **海外远程兽医平台提供模式参考。** FirstVet、Vetster、Pawp 说明远程兽医、订阅会员、连续护理具备商业模式参考价值。
6. **本产品的差异化方向不应是“另一个 AI 宠物医生聊天框”。** 真正机会是：AI 多模态分诊入口 + 宠物健康档案 + 急症红旗识别 + 执业兽医承接 + 线下医院推荐 + 用药复诊提醒 + 智能硬件健康数据解释层。

## 1. 分析方法与数据来源

### 1.1 使用的分析方法

| 方法 | 用途 | 本报告中的应用 |
|---|---|---|
| 竞品分层法 | 区分直接竞品、间接竞品、替代竞品 | 将竞品分为线上问诊、电商健康、线下医院、智能硬件、内容社区、通用大模型 |
| 功能矩阵 Benchmarking | 横向比较功能完整度 | 比较 AI 问诊、兽医承接、医院转诊、档案、提醒、硬件等能力 |
| 战略群组分析 | 判断不同竞品所处竞争区域 | 按“医疗承接能力”和“日常健康管理能力”绘制竞品分布 |
| 波特五力模型 | 判断行业竞争强度 | 分析现有竞争者、潜在进入者、替代品、供应商、用户议价能力 |
| 价值曲线 / 蓝海战略 | 寻找差异化空间 | 比较不同玩家在关键价值维度上的强弱 |
| SWOT / TOWS | 制定竞争策略 | 输出增长型、防御型、补短板、避险策略 |
| 商业模式画布思想 | 分析竞品变现方式 | 拆解问诊、会员、电商、医院导流、硬件、保险等收入来源 |

## 2. 竞品边界与竞品分层

### 2.1 本产品所处价值链

```text
宠物建档
↓
日常健康记录
↓
AI 多模态分诊
↓
急症风险识别
↓
线上兽医问诊
↓
线下医院转诊
↓
用药 / 喂食 / 复诊提醒
↓
药品用品履约
↓
智能硬件预警
```

因此，竞品不能只看“有没有 AI 宠物医生”，而要看谁占据了这条链路中的关键节点。

### 2.2 竞品分类

| 类型 | 代表 | 竞争关系 | 竞争焦点 |
|---|---|---|---|
| 直接竞品 | 京东健康宠物医院、JDH 宠物医生、海外 FirstVet / Vetster / Pawp | 直接解决线上宠物问诊和健康咨询 | 问诊、兽医、AI、线上服务 |
| 专业医疗竞品 | 新瑞鹏、瑞派、安安宠医、宠颐生等 | 承接线下诊疗和复杂病例 | 医院、医生、检查、手术、专科 |
| 智能硬件竞品 | PETKIT 小佩、智能摄像头、猫砂盆、饮水机、喂食器 | 占据宠物行为和健康数据入口 | 设备数据、远程监控、异常提醒 |
| 电商履约竞品 | 京东、天猫、美团、Chewy、波奇 | 占据药品、用品、粮食、处方粮履约 | 商品、供应链、复购、配送 |
| 内容社区竞品 | 小红书、抖音、百度、知乎、微信群 | 占据用户第一次搜索行为 | 内容、经验、种草、流量 |
| 通用 AI 竞品 | ChatGPT、豆包、通义、Kimi、DeepSeek 等 | 替代轻量问答 | 泛问答、多模态、低成本 |

## 4. 核心竞品画像

### 4.1 京东健康宠物医院：最强直接竞品

京东健康宠物医院定位为宠物互联网医疗与健康服务平台，核心模式是：

```text
AI / 线上问诊
+ 执证宠医
+ 药品和营养品
+ 上门检测
+ 宠物保险
+ 线下医院转诊
```

公开报道显示，京东健康构建了“医、检、药、险”一站式服务体系，并联动全国超过 **11000 家** 正规线下宠物医院，推行“轻症线上问诊、重症线下转诊”。另有报道提到，京东健康宠物医院依托 **5500+ 执业兽医师**，提供 7×24 小时在线服务，平均 8 秒接诊；其 AI 宠物医生由京东自研 AI 医疗模型打造。

| 维度 | 能力 |
|---|---|
| 医生资源 | 5500+ 执业兽医师，7×24 小时在线 |
| 响应效率 | 平均 8 秒接诊 |
| AI 能力 | 自研 AI 医疗模型支持 AI 宠物医生 |
| 线下资源 | 联动全国 11000+ 正规线下宠物医院 |
| 商品履约 | 京东药品、营养品、处方食品、保险 |
| 服务链路 | 科普、评估、问诊、干预、药品、保险 |
| 品牌优势 | 京东平台信任、供应链、支付、履约能力 |

#### 商业模式

| 收入来源 | 说明 |
|---|---|
| 线上问诊 | 图文 / 视频 / 极速问诊 |
| 药品与营养品 | 驱虫药、处方食品、治疗药、保健品 |
| 宠物保险 | 医疗险、直赔 |
| 检测服务 | 上门检测、报告解读 |
| 品牌合作 | 宠物药品、处方粮、营养品首发 |
| 平台交易佣金 | 京东电商体系内交易转化 |

#### 优缺点

| 类型 | 分析 |
|---|---|
| 优势 | 医生多、响应快、供应链强、品牌信任高、线下医院资源广、药品保险闭环强 |
| 劣势 | 用户心智更偏“买药 / 电商 / 京东服务”，日常健康记录、慢病管理和多模态长期档案未必是第一心智 |
| 对本产品威胁 | 线上问诊、药品履约、医院导流能力强，若进一步完善宠物健康档案，会成为全链路强竞品 |
| 差异化空间 | 避开正面拼电商，主打“AI 多模态分诊 + 长期健康档案 + 用药复诊提醒 + 硬件数据预警” |

### 4.2 新瑞鹏宠物医疗集团：线下医疗承接型竞品

新瑞鹏是以宠物医疗为主营的大型综合性企业集团。官方资料显示，新瑞鹏在全国范围内开设宠物中心医院、国际医院、专科医院及社区医院 **1000 多家**，覆盖 **100 多个城市**，并重点打造 15 大宠物专科。

| 维度 | 能力 |
|---|---|
| 线下医院 | 1000+ 门店 |
| 城市覆盖 | 100+ 城市 |
| 专科能力 | 15 大宠物专科 |
| 诊疗能力 | 检查、诊断、治疗、手术、住院 |
| 品牌集群 | 美联众合、芭比堂、宠颐生、瑞鹏、安安宠医等 |
| 用户场景 | 急症、专科、复杂疾病、线下治疗 |

#### 优缺点

| 类型 | 分析 |
|---|---|
| 优势 | 真实医疗能力强，能处理复杂病情、急症和线下检查 |
| 劣势 | 日常高频健康管理触达弱，线上 AI 分诊、档案提醒、硬件预警不是核心优势 |
| 对本产品威胁 | 其线下诊疗能力不可替代，若自建线上入口，会与本产品形成直接竞争 |
| 差异化空间 | 本产品可做医院前置入口：AI 分诊、病情摘要、复诊提醒、用药管理，医院做确诊和治疗 |

### 4.3 PETKIT 小佩：智能硬件数据入口型竞品

PETKIT 是智能宠物用品品牌，覆盖智能猫砂盆、喂食器、饮水机、摄像头等产品。官方应用介绍显示，PETKIT App 服务全球超过 **600 万** 宠物家长，可 24/7 跟踪猫砂盆使用、进食、饮水和活动，并帮助用户发现细微行为变化。

| 维度 | 能力 |
|---|---|
| 用户规模 | 600 万+ 全球宠物家长 |
| 数据类型 | 饮水、进食、猫砂盆、活动、体重、行为 |
| 硬件入口 | 猫砂盆、饮水机、喂食器、摄像头 |
| 使用频率 | 高，设备每日采集 |
| 健康潜力 | 可识别饮水、排尿、进食、活动异常 |
| 主要短板 | 缺少兽医诊疗、医院转诊、处方药合规闭环 |

#### 优缺点

| 类型 | 分析 |
|---|---|
| 优势 | 掌握连续行为数据，日常使用频率高，硬件粘性强 |
| 劣势 | 医疗解释能力和兽医承接能力弱，难以独立完成诊疗服务 |
| 对本产品威胁 | 若硬件厂商向健康管理和 AI 问诊延伸，会抢占数据入口 |
| 差异化空间 | 本产品可与硬件合作，做“硬件数据健康解释层 + 兽医承接层” |

### 4.4 Chewy：海外电商 + 复购 + 健康服务参考样本

Chewy 是美国宠物电商和宠物健康服务平台。其 2025 财年财报显示，Chewy 2025 财年净销售额为 **126.0 亿美元**，同比增长 **6.2%**，活跃客户达到 **2130 万**。其 2024 财年 Autoship 自动订购销售额占全年净销售额 **79.2%**，说明宠物消费天然适合周期复购和订阅式服务。

| 维度 | 能力 |
|---|---|
| 电商规模 | 2025 财年净销售额 126.0 亿美元 |
| 用户规模 | 2130 万活跃客户 |
| 订阅复购 | Autoship 占 2024 财年净销售额 79.2% |
| 商品能力 | 宠物食品、用品、药品、健康商品 |
| 健康服务 | 宠物健康内容、兽医相关服务、保险与健康计划 |
| 主要启发 | 宠物健康和用品适合“周期提醒 + 自动复购 + 会员” |

**对本产品启发：** 如果只做 AI 问诊，留存会低；如果把驱虫、用药、处方粮、护理用品、主粮、复诊做成周期提醒和复购链路，商业价值会明显提升。

### 4.5 FirstVet / Vetster / Pawp：海外远程兽医模式参考

| 平台 | 核心模式 | 启发 |
|---|---|---|
| FirstVet | 在线视频兽医服务、24/7 护理、必要时推荐线下诊所 | 远程兽医需与线下就医边界结合 |
| Vetster | 订阅模式，含虚拟兽医预约、无限消息咨询、多宠覆盖 | 会员与多宠管理可行 |
| Pawp | 24/7 在线兽医和护士团队，持续护理体验 | 连续护理比单次问诊更重要 |

**海外模式启发：** 宠物健康服务不应只靠单次问诊，而应走远程兽医、follow-up、订阅会员、多宠管理和持续护理。

### 4.6 内容社区与通用大模型：替代型竞品

小红书、抖音、百度、知乎、微信群等平台不是宠物医疗产品，但它们是用户遇到宠物异常时的第一反应入口。其优势是流量大、内容丰富、搜索方便；劣势是医学质量参差不齐。

通用大模型可以回答宠物健康问题，也可以识别图片、总结信息，但通常缺少宠物长期档案、急症红旗规则、执业兽医承接、线下医院推荐、用药复诊提醒、处方药合规闭环和医生反馈数据。因此，通用大模型会替代轻量问答，但难以替代完整宠物健康管理平台。

## 11. 差异化机会

### 11.1 从“AI 问答”升级为“AI 分诊”

通用大模型能回答问题，但难以稳定完成：

```text
识别宠物档案
↓
按症状路径追问
↓
命中急症红旗
↓
输出风险等级
↓
转兽医 / 转医院
↓
生成诊后提醒
```

因此，本产品的第一差异化不是“AI 会回答”，而是“AI 能把宠物健康问题转成可执行流程”。

### 11.2 从“问诊一次”升级为“长期健康管理”

真正有长期价值的是：

- 用药提醒；
- 疫苗提醒；
- 驱虫提醒；
- 复诊提醒；
- 体重记录；
- 粪便记录；
- 饮水记录；
- 慢病管理。

### 11.3 做医生工作台，提高兽医效率

本产品可以提供：AI 病情摘要、宠物历史档案、症状时间线、图片 / 语音资料、急症红旗、用户最关心的问题、用药复诊建议模板。

### 11.4 接入硬件数据，而不是自研硬件

```text
智能猫砂盆数据
+ 饮水机数据
+ 喂食器数据
+ 摄像头片段
↓
AI 健康解释
↓
异常预警
↓
兽医承接
```

### 11.5 合规反而可以成为壁垒

产品必须避免：AI 确诊、AI 开药、AI 推荐处方药剂量、社区互相开药、为卖货制造医疗焦虑。合规不是束缚，而是长期信任壁垒。

## 12. 竞品分析结论

综合数据和竞品能力对比，当前宠物健康管理与宠物诊疗数字化赛道已经形成多类玩家并行竞争的格局：

1. 京东健康是当前最强直接竞品。它拥有平台流量、执业兽医、7×24 小时在线问诊、快速接诊、药品营养品供应链、保险和线下医院联动能力。它的短板不是资源，而是用户心智更偏电商与问诊服务，长期日常健康管理和智能硬件数据解释仍有进一步深化空间。
2. 新瑞鹏等线下医院集团掌握最终诊疗能力。它们在线下检查、手术、住院和专科医疗方面不可替代，但线上分诊、诊前病情整理、诊后复诊管理、日常提醒和用户长期触达仍存在产品化空间。
3. PETKIT 等智能硬件厂商掌握高频数据入口。它们能持续获取宠物饮水、进食、活动、如厕等行为数据，具备主动健康预警潜力，但缺少执业兽医、线下医院和合规诊疗闭环。
4. 海外 FirstVet、Vetster、Pawp 证明远程兽医和连续护理具备商业模式参考价值。它们的重点不是单次回答，而是 24/7 远程兽医、follow-up、订阅会员、多宠管理和持续护理。
5. 内容社区和通用大模型会替代低价值轻问答。如果本产品只做“宠物问题 AI 聊天框”，会很快被通用大模型、小红书、抖音、百度和京东这类平台替代。

因此，本产品的最优定位不是：

> AI 宠物医生，替代宠物医院。

而应该是：

> 面向宠主的 AI 宠物健康管理与辅助分诊平台，以多模态 AI 分诊为入口，以宠物健康档案为核心，以执业兽医和线下医院为安全承接，以用药复诊提醒和硬件数据预警形成长期健康管理闭环。

最终竞争策略可以概括为：

```text
不和京东正面拼药品履约
不和新瑞鹏正面拼线下诊疗
不和 PETKIT 正面拼硬件制造
不和通用大模型正面拼泛问答

而是做：
AI分诊 + 档案 + 兽医 + 医院 + 提醒 + 硬件健康解释
这一条完整闭环。
```

一句话结论：

> 这个赛道的竞争已经开始，但真正饱和的是“问答、电商、医院、硬件”这些单点；尚未饱和的是“可信的 AI 宠物健康管理闭环”。
