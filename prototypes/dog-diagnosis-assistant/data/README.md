# Data

The original training datasets are not included in this clean package.

Reasons:

1. They may contain copyrighted content derived from books or PDFs.
2. Public repositories should not upload datasets unless the license is clear.
3. Keep large or sensitive data outside Git and document how to reproduce it.

Recommended local layout:

```text
data/
├── dataset.json
└── cleaned_dataset.json
```

These files are ignored by `.gitignore` by default.
